# Amigo Secreto 🎁

Aplicación web para organizar sorteos de Amigo Secreto de forma simple 
y aleatoria, garantizando que ningún participante se repita en el sorteo.

## ¿Cómo funciona?

1. Ingresa los nombres de los participantes uno por uno
2. Haz clic en el botón para generar el sorteo
3. El sistema asigna aleatoriamente un Amigo Secreto a cada participante
4. Sin repetidos y sin que nadie quede sin asignar

## Características

- Agrega tantos participantes como desees
- Sorteo aleatorio sin repeticiones
- Interfaz simple e intuitiva
- Sin necesidad de registro ni backend
